🏥 Hospital Queue Management System

A full-stack web app that simplifies hospital queue management, helping patients track their turn and enabling staff to manage queues efficiently.

🚀 Live Demo

View Here   https://queueheydaraa.netlify.app

Features

Users: Register, login, join queue, track position,look thier turn on public screen.
Admin/Staff: Manage queue, call next patient, monitor status
Tech Stack
Frontend: React, HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MySQL
Other: JWT Authentication, REST API


Installation

git clone https://github.com/Heydaraa/hospital_queue_system.git

cd backend

nodemon server.js

npm start

cd ../clien
t
npm install

npm run dev


Set your .env with DB credentials and JWT_SECRET.
